# Постоянная ссылка (телефон)

Этот проект — статический сайт. Чтобы была ссылка, ПК должен раздавать страницу (сервер).

## Вариант A (проще): постоянная ссылка в вашей Wi‑Fi сети

1) Запускайте сервер: `start-server.bat`
2) Сделайте IP ПК постоянным в роутере:
   - DHCP Reservation / “Привязать IP к MAC”
   - например закрепите `192.168.173.76`
3) С телефона (в той же Wi‑Fi сети) открывайте:
   - `http://192.168.173.76:5173/`

Чтобы ссылка работала “всегда”, добавьте `start-server.bat` в автозагрузку Windows.

## Вариант B: постоянная ссылка из любого интернета (рекомендую)

Самый простой способ без проброса портов — **Tailscale**.

1) Установите Tailscale на ПК и телефон, войдите в один аккаунт.
2) Включите “MagicDNS” в админке Tailscale (по желанию).
3) Сервер запускайте: `start-server.bat`
4) С телефона открывайте (стабильно, из любого места, пока включён ПК):
   - `http://<tailscale-ip-пк>:5173/`
   - или `http://<имя-пк>:5173/` (если включён MagicDNS)

## Важно про “аккаунт/прогресс”

Сейчас данные хранятся в браузере (localStorage) — на каждом устройстве отдельно.
Если нужно, чтобы один аккаунт и прогресс были общими между ПК/телефоном, нужен серверный бэкенд (API + база).
